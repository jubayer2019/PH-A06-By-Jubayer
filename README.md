# 🚀 DigiTools - Premium Digital Marketplace

**DigiTools** is a modern, high-performance digital marketplace platform built for professionals who need quick access to premium AI tools, design assets, and productivity software.

## 🛠️ Technology Stack

This project leverages the latest web technologies to ensure a smooth and responsive user experience:

  * **Frontend Framework:** React.js (Functional Components & Hooks)
  * **Styling:** Tailwind CSS (Utility-first CSS)
  * **UI Components:** DaisyUI (Tailwind-based component library)
  * **State Management:** React `useState` & `useContext`
  * **Icons:** FontAwesome 6
  * **Deployment:** Netlify
  * **Notifications:** React Hot Toast

-----

## ✨ Key Features

### 1\. Dynamic Shopping Cart System

Experience a seamless flow from browsing to buying. The cart features a real-time "Indicator" badge in the navbar and an inline count within the navigation tabs. Users can add products, remove them, and see an instant calculation of their total before proceeding to a virtual checkout.

### 2\. Tabbed Content Interface

The platform uses a sleek, radio-input-based tab system that allows users to switch between the **Product Gallery** and their **Personal Cart** without page reloads. This provides a "Single Page Application" (SPA) feel that keeps the workflow fast.

### 3\. Fully Responsive & Animated UI

Designed with a "Mobile-First" approach, the layout adapts perfectly to any screen size—from smartphones to ultra-wide monitors. It features custom gradient buttons, hover-transformed cards, and entrance animations to make the browsing experience feel premium.
